# 📊 YouTube Trending Video Analysis Dashboard

## 📌 Project Overview

This project analyzes YouTube trending video data across multiple regions (India, US, Russia) using Power BI.
The dashboard provides insights into views, likes, comments, and category-wise performance.

---

## 🎯 Objectives

* Analyze trending video performance
* Compare engagement across regions
* Identify top categories and videos
* Understand trends over time

---

## 📁 Dataset

The dataset includes:

* Video ID
* Title
* Trending Date
* Category ID
* Views
* Likes
* Dislikes
* Comment Count

Data was cleaned and reduced for optimization.

---

## 📊 Dashboard Features

* Total Views, Likes, Comments, Video Count (Cards)
* Region filter (India, US, Russia)
* Top 10 videos by views
* Category-wise distribution (Pie chart)
* Views trend over time
* Views vs Likes relationship (Scatter plot)

---

## 🛠️ Tools Used

* Power BI
* Excel (Data Cleaning)

---

## 📌 Key Insights

* Entertainment and Music categories dominate views
* Higher views generally correlate with higher likes
* Regional trends vary significantly
